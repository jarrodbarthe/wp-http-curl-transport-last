This WordPress plugin ensures that the HTTP cURL transport is used as last.

== Description ==

This plugin can fix the following error caused in cURL v7.31.0:

`SSL read: error:00000000:lib(0):func(0):reason(0), errno 0`

See for information the following pages:

*	http://wordpress.org/support/topic/ssl-read-error00000000lib0func0reason0-errno-0
*	http://sourceforge.net/p/curl/bugs/1249/


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your 
WordPress installation and then activate the Plugin from Plugins page.
